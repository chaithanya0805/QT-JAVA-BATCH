
Control Statements 
===================
Control statement enables a programmer to control flow of a program.

Control statement allows us to make decisions, we can execute the code repeatedely and we can jump from one section of code to another section.

In java, we have three control statements.

1) Decision making statement 

2) Iteration statement 

3) Jump statement 

1) Decision making statement 
-----------------------------
It is used to declare the conditions in our program.

Decision making statement is possible by using following ways.

i) if stmt 

ii) if else stmt 

iii) if else if ladder 

iv) nested if stmt 

v) switch case 


i) if stmt 
============
It executes the source code only if our condition is true.

syntax:
------
	if(condition)
	{
		-
		- //code to be execute 
		-
	}

ex:
---
class Test 
{
	public static void main(String[] args) 
	{
		System.out.println("stmt1");	
		
		if(!(5>20))
		{
			System.out.println("stmt2");
		}
		
		System.out.println("stmt3");
	}
}

ii) if else stmt 
================
It executes the source code either our condition is true or false.

syntax:
------
	if(condition)
	{
		- //code to be execute if cond is true 
	}
	else
	{
		- //code to be execute if cond is false 	
	}

iv) nested if stmt

syntax:
------

	{

		{
			-
			- //code to be execute 
			-
		}
	}

class Test 
{
	public static void main(String[] args) 
	{

			if(false)
			{
				System.out.println("stmt3");
			}
			
			System.out.println("stmt4");
		}
		
		System.out.println("stmt5");
	}
}

v) switch case 
==============
It is used to execute the source code based on multiple conditions.
It is similar to if else if ladder.
We need to use switch case , when we want to take options from the enduser.

ex:
	switch(expression/condition)
	{
		case value1 : //code to be execute
			      break stmt;	

		case value2 : //code to be execute
			      break stmt;

		-
		-
		default :  //code to be execute 
	}

ex:
---
import java.util.Scanner;
class Test 
{
	public static void main(String[] args) 
	{
		Scanner sc = new Scanner(System.in);
		
		System.out.println("Enter the option :");
		int option  = sc.nextInt();
		
		switch(option)
		{
			case 100 : System.out.println("It is a police number");
					   break;
			case 103 : System.out.println("It is a enquiry number");
					   break;
			case 108 : System.out.println("It is a emergency number");
					   break;
			default  : System.out.println("Invalid option");
		}
		
		
	}
}

Declaration of break statement is always optional.If we won't define break statement then from where our condition is satisified from there all cases will be executed that state is called "fall through state" of switch case.

yield
======
A yield keyword introduced in Java 14.

It is used to return a value from a case block within a switch expression.

ex:
---
import java.util.Scanner;
class Test 
{
	public static void main(String[] args) 
	{
		Scanner sc = new Scanner(System.in);
		
		System.out.println("Enter the Day of a Week :");
		int day = sc.nextInt();//4
		
		String result = switch(day)
		{
			case 1,2,3,4,5 -> { yield "Week Days"; }
			default  -> { yield "Weekends"; }
		};
		System.out.println(result);
	}
}
