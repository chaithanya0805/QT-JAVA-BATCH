Q) Write a java program to display extension of a file?

input:
	abc.txt 
output:
	txt 


class Test  
{
	public static void main(String[] args) 
	{
		String str = "abc.txt";
		
		int index = str.indexOf('.');
		
		String result = str.substring(index+1);
		
		System.out.println(result);
	}
}

Q) Write java program to check given maid id is valid or not?

input:
	raja@gmail.com
output:
	valid 


class Test  
{
	public static void main(String[] args) 
	{
		String str = "raja@gmail.com";
		
		int atPosition = str.indexOf('@');
		
		int dotPosition = str.lastIndexOf('.');
		
		if(atPosition!=-1 && dotPosition !=-1 && atPosition<dotPosition)
			System.out.println("Valid");
		else
			System.out.println("Invalid");
	}
}

Q) Write a java program to find out maximum integer value from given string?

input:
	1kg of mango is 100 rupees and 2kg of apple is 200 rupees
output:
	200


class Test  
{
	public static void main(String[] args) 
	{
		String str="1kg of mango is 100 rupees and 2kg of apple is 200 rupees";
		String[] sarr = str.split("\\D+");
		
		int max = Integer.MIN_VALUE; //-2147483648 
		
		for(String s : sarr)
		{
			int n = Integer.parseInt(s);
			
			if(n>max)
			{
				max = n;
			}
		}
		System.out.println(max);
	}
}

Q) Write a java program to find out given string is palindrome or not?

input:
	racar 
output:
	It is a palindrome string 


class Test  
{
	public static void main(String[] args) 
	{
		String str="racar";
		
		String rev = "";
		
		//reading reverse 
		for(int i=str.length()-1;i>=0;i--)
		{
			rev += str.charAt(i);
		}
		
		if(str.equals(rev))
			System.out.println("It is a palindrome string");
		else
			System.out.println("It is not a palindrome string");
	}
}

Q) Write a java program to find out the string starting with uppercase letter?

Q) Jack purchased a new laptop for his office work. His son started learning how to frame the words using alphabets. Jack wants to write a program to find out longest word created by his son.

input:
	apple cat banana elephant dark 
output:
	elephant 


class Test  
{
	public static void main(String[] args) 
	{
		String str="apple cat banana elephant dark";
		
		String[] sarr = str.split(" ");	
		
		int size = sarr[0].length();
		String word = "";
		
		for(String s : sarr)
		{
			if(s.length() > size)
			{
				size = s.length();
				word = s;
			}
		}
		System.out.println(word);
	}
}
