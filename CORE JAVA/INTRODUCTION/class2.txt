> Standalone app
----------------
It is a normal java program which contains main method is called standalone application.

ex:
	class Test
	{
		public static void main(String[] args)
		{	
			-
			- //code here 
			-
		}
	}

> Desktop app
-------------
It is a software application or a program specially designed to perform perticular task.
ex:
	Control Panel 
	Recycle Bin 
	VLC Media Player 
	and etc.

> Two-tier app 
---------------
Having more then one tier is called two-tier application.

Diagram: class2.2


> Distributed app
-----------------
Diagram: class2.3

In client/server architecture ,if multiple requests go to main server then main server will distribute those requests to it's parallel servers to reduce the burdun of main server such type of application is called distributed application. 


> Enterprise app
----------------
Application which deals with large complex business logic with the help of middleware services is called enterprise application.

Here middleware services means authentication, authorization, malware production ,security and etc.

ex:
	Online shopping website
	Skype 
	Zoom meeting 
	and etc.

> ERP app 
-----------
ERP stands for Enterprise Resource Planning.
ERP is used to maintain the data in a enterprise.

> N-tier app 
------------
Having more then two tier is called N-tier application.


Mobile App 
-----------
It is a software application or a program specially designed for wireless network devices like phone , tab, cell, cellular and etc.
ex:
	PubG
	Gpay 
	Temple Run 
	and etc.
